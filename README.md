**React Single Page Layout**

**Project Overview**

This project demonstrates a simple React Single Page Application (SPA) built using React CDN, JSX, reusable components, and CSS styling. It showcases how to structure a clean and modular React UI without using build tools.


**Technologies Used**

HTML5
CSS3
JavaScript
React.js
ReactDOM
Babel


**Components**

The application is divided into reusable components:

**Navbar Component**
Displays the website title and navigation links: Home, About, Services, Contact.

**Hero Component**
Welcomes users with a heading and introduction to React.js.

**Card Component**
Reusable UI card used in the Services section.

**Services Component**
Displays services like:
Web Development
UI/UX Design
React Application Development

**Footer Component**
Shows copyright information.

**App Component**
Main component that combines all sections.


**Features**
Reusable Components
Responsive Card Layout
Clean UI Design
Single Page Structure
Easy to Customize


**Output**

(Simple React SPA layout with navbar, hero section, services cards, and footer.)


**Project Structure**

/project-folder
│── index.html
│── style.css
│── app.js


**How to Run**

Download or clone the repository
Open index.html in your browser
Make sure you have internet (for React CDN)


**Conclusion**

React components help build modular, reusable, and maintainable web applications efficiently.
